## 二、V1 产品定位（已实现）

### 2.1 一句话定位

OVOFORGE 是一个**手绘风格的结构化玩法知识库**，把每个游戏玩法拆解为：规则说明 → 关键代码 → 可试玩 Demo → 设计文章。

### 2.2 内容模型（四层体系）

```
├─ 玩法帖子（Play）
│   ├─ 标题 / 副标题 / 标签 / 难度 / 技术栈
│   ├─ 玩法拆解（结构化 JSON）
│   ├─ 关键代码片段
│   ├─ Demo（iframe / video / 母型回退）
│   └─ 文章（MDX）
│
├─ 母型玩法（Archetype）
│   └─ 12 个固定母型，作为分类锚点
│
├─ 核心编辑器原型（Prototype）
│   └─ 5 个可复用编辑器/交互原型
│
└─ 玩法特征（Feature）
    └─ 9 个通用玩法特征标签
```
