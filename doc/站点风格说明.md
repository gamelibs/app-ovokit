# OVOKIT 手绘风格设计系统实施方案 V1.0

## 项目目标

为 OVOKIT（游戏玩法技术实现分享站）建立一套统一的手绘风格设计系统。

核心特点：

* 首页为玩法技术内容聚合入口
* 详情页为玩法技术拆解与实现参考中心
* 全站统一手绘品牌风格
* AI 可辅助生成结构化图解、封面、流程图（可选）
* 所有装饰性视觉素材优先使用 SVG
* 保持极高页面性能

# 六、RoughJS使用规范

官网：

https://roughjs.com

安装

```bash
npm install roughjs
```

创建矩形

```javascript
import rough from 'roughjs';

const rc = rough.svg(svg);

rc.rectangle(
  10,
  10,
  200,
  100,
  {
    roughness:2,
    bowing:1,
    stroke:'#222',
    fill:'none'
  }
);
```

---

参数规范

```javascript
roughness:2
bowing:1
strokeWidth:2
```

禁止超过：

```javascript
roughness:4
```

否则影响阅读。
